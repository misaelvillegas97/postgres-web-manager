Preguntas frecuentes

¿Puedo usar el modo Self-Hosted en producción en mi empresa?

Sí. PgStudio se distribuye bajo la licencia Apache 2.0. Puedes usarlo en producción, modificarlo y redistribuirlo libremente.

¿Mis credenciales de PostgreSQL se envían al servidor de PgStudio Cloud?

Sí, el gateway necesita la contraseña para conectarse a PostgreSQL via TCP. Todas las credenciales guardadas se cifran con AES-GCM usando CREDENTIALS_ENCRYPTION_KEY. Nunca se almacenan en texto plano.

¿Puedo conectar bases de datos en una VPC privada?

En el modo Cloud, la base de datos debe ser accesible desde nuestra infraestructura. Para VPCs privadas, el modo Self-Hosted dentro de tu red es la opción correcta.

¿Qué versiones de PostgreSQL son compatibles?

PgStudio usa node-postgres (pg) y el protocolo estándar de PostgreSQL. Compatible con PostgreSQL 12+ y cualquier PostgreSQL-compatible (como Amazon RDS, Neon, Supabase, CockroachDB en modo PG).
